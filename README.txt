To compile the code:
Just simply run on App Engine through the google cloud icon. Then can
easily access by going to localhost:8080 in a browser.

To deploy the app:
Right click the project  Project—Properties—GoogleCloudPlatform—AppEngineDeployment,
in the account drop down, select your google accountto use then select the project 
have created previously. Follow the create an app engine app in the cloud console and 
select region you prefer.

Caution: Remember to make sure JRE system library is using JDK

Finally access app live at http://[appid].appspot.com
